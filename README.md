# pinterest
Proyecto Pinterest Usando JSON, Node.JS, BEM, SASS

### Requerimientos Técnicos

El proyecto fue realizado con las siguientes dependencias NPM:

+ Dependencias de Desarrollo 
  - Gulp `~v.3.9.1`
  - Gulp-Concat `~v.2.6.1`
  - Gulp-Jshint `~v.2.0.4`
  - Gulp-Minify-CSS `~v.1.2.4`
  - Gulp-Sass `~v.3.0.0`
  - Gulp-Uglify `~v.2.0.0`

+ Dependencias de Producción 
  - jQuery `~v.3.1.1`
  - Bootstrap `~v.3.3.7`

### Levantamiento ambiente de desarrollo

+ Clonar repositorio.
+ Posicionado en nuestro proyecto hacer correr `npm install`para descargar e instalar todas las dependencias utilizadas.
+ Correr gulp para que se realizen las tareas especificadas.
+ Hcaer el HTML con el nav + el div que contiene todas las fotos
+ Agregar el Json a la variable data 
+ Agregar las funciones para recorrer el Json en el archivo Main.js
+ Agregar el modal
